项目简介

Demoon UI 是一个个人练习制作的UI框架，适合移动端和PC端使用。

提供的组件

Demoon UI 提供了按钮、输入框、栅格、布局、弹出层、标签栏、折叠栏等常用组件。

项目特点

Demoon UI 的组件有丰富的单元测试，测试覆盖率超过 90%，同时使用了 Travis CI 进行持续集成。
